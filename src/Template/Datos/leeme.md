Índice de Situación Académica (ISA):

El Índice de Situación Académica (ISA) se calcula ponderando la última nota obtenida en cada materia (aprobadas o reprobadas, sin incluir las retiradas) que aparecen en el módulo de Situación Académica.

ISA =  ∑ Crédito X Ultima Nota /  ∑ credito

Índice de Rendimiento Académico (IRA):

El Índice de Rendimiento Académico (IRA) o también llamado Índice de Proceso o Índice de Grado se calcula ponderando todas las notas aprobadas y reprobadas (sin incluir las retiradas) que aparecen en el módulo de Histórico.

IRA =  ∑ Crédito X Nota /  ∑ credito
